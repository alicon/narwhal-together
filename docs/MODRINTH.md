# NARwhal Together

**Little tools for big Minecraft adventures together.**

NARwhal Together is a family-focused Fabric mod that helps young players spend less time typing commands and more time playing together.

## Current feature: quick teleport

Use one configurable action to teleport to another online player—no chat commands and no operator permissions required.

- One other player online: teleport directly to them.
- Several players online: cycle through them alphabetically.
- Controller friendly: bind the action directly or add it to Controlify's radial menu.
- Family safe: spectators are skipped and a short cooldown prevents accidental repeated teleports.
- Dimension aware: regroup across the Overworld, Nether, and End.

## Installation

NARwhal Together must be installed on every participating client and on the server or LAN host.

### Required

- Fabric Loader
- Fabric API
- Minecraft Java Edition 1.21.11

### Optional

- Controlify for controller support and radial-menu access

The default keyboard binding is `G` and can be changed under **Options → Controls → Key Binds → NARwhal Together**.

## Why “NARwhal”?

NAR stands for Noelle, Aoife, and Rufus—the three young players who inspired the project. Future features will continue to focus on making shared Minecraft adventures welcoming, surprising, and easy for families.
