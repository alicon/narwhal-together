# Cops and Robbers 0.1.0

Initial alpha release for Minecraft 1.21.11 on Fabric.

## Features

- Adds the Cops and Robbers mod with a dedicated `cops_robbers` namespace.
- Adds a rideable police cruiser with lights, siren, controller shortcuts, and creative flight tricks.
- Adds a rideable fire truck for emergency scenes.
- Adds bank robbers, cops, bank tellers, and firefighters.
- Adds robber capture behavior for police cruiser patrols.
- Adds placeable police station and bank kits.
- Adds creative spawn eggs for all new entities.

## Requirements

- Minecraft Java Edition 1.21.11
- Fabric Loader
- Fabric API
