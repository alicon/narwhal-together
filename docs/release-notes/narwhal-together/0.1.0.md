# NARwhal Together 0.1.0

Initial alpha release for Minecraft 1.21.11 on Fabric.

## Features

- Adds a controller-friendly quick teleport action for regrouping with another online player.
- Teleports directly when exactly one eligible player is online.
- Cycles through eligible players alphabetically when several players are online.
- Skips spectators and includes a short cooldown to avoid accidental repeated teleports.
- Works across dimensions so families can regroup from the Overworld, Nether, or End.
- Includes optional Controlify integration for controller and radial-menu workflows.

## Requirements

- Minecraft Java Edition 1.21.11
- Fabric Loader
- Fabric API
