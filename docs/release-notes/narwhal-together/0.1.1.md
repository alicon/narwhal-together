# NARwhal Together 0.1.1

Compatibility release for Minecraft 1.21.11 and 1.21.1 on Fabric.

## Updates

- Publishes separate artifacts for supported Minecraft versions.
- Keeps the controller-friendly teleport behavior from the initial alpha release.

## Requirements

- Minecraft Java Edition 1.21.11 or 1.21.1
- Fabric Loader
- Fabric API
