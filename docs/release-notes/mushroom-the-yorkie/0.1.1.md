# Mushroom the Yorkie 0.1.1

Bugfix release for Minecraft 1.21.11 on Fabric.

## Fixes

- Fixes Mushroom visually standing when ordered to sit.
- Keeps the sitting pose synced while Mushroom is ordered to sit, even when daytime sleep checks run.
- Prevents the indoor potty warning behavior from overriding an explicit sit command.

## Requirements

- Minecraft Java Edition 1.21.11
- Fabric Loader
- Fabric API
