# Mushroom the Yorkie 0.1.0

Initial alpha release for Minecraft 1.21.11 on Fabric.

## Features

- Adds Mushroom, a tiny Yorkie companion with custom model, texture, spawn egg, and treat item.
- Supports one Mushroom per player by default, including creative spawn eggs.
- Adds configurable wake-up spawning after players sleep.
- Mushroom follows his owner, accepts treats, and can be called back with a treat in hand.
- Mushroom barks when he has been indoors too long and needs to go outside.
- At night indoors, Mushroom curls up to sleep, closes his eyes, and can be stirred briefly by double-clicking.
- Mushroom can bark at and chase peaceful mobs, then settle down for the day when called off with a treat.
- Creative flying support lets Mushroom follow with loop-de-loops and barrel rolls.
- Gentle landing behavior matters: dropping too fast can injure little Mushroom.

## Requirements

- Minecraft Java Edition 1.21.11
- Fabric Loader
- Fabric API
